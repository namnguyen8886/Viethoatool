💡 Hướng dẫn

Luồng sử dụng:
1️⃣ Upload file zip chứa folder /plugins
2️⃣ Hệ thống quét và gợi ý plugin cần dịch
3️⃣ Tích chọn plugin muốn dịch → Lưu
4️⃣ Quay lại tab Dịch File → dịch với filter plugin

● NÊN ĐÀO — Plugin có file dịch được
● CẨN THẬN — Cần xem lại trước khi dịch
● BỎ QUA — Library, không cần dịch
